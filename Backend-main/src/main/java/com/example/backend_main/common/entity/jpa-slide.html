Q. JPA란?

  • JPA는 자바 객체와 DB 테이블을 연결해 데이터를 처리하는 기술이다.
  • Ai-Law 프로젝트에서는 고객 문의 기능에 적용하여 등록, 조회, 수정, 삭제를 구현했다.
  • Repository를 통해 SQL 작성 부담을 줄이고 객체 중심으로 데이터를 관리할 수 있다.
예시 코드
public interface CustomerInquiryRepository
    extends JpaRepository<CustomerInquiry, Long> {

    List<CustomerInquiry>
    findByWriter_UserNoOrderByCreatedAtDesc(Long userNo);
}

return customerInquiryRepository.save(inquiry);
customerInquiryRepository.findById(id);

핵심 구현 내용

  • save() : 문의 등록 및 수정 처리
  • findById() : 문의 상세 조회
  • findByWriter_UserNoOrderByCreatedAtDesc() : 내 문의 목록 조회
  • 로그인 사용자 기준으로 본인 문의만 조회·수정·삭제 가능
